## Data Audit API

API orientada a dados, para auditoria de inconsistências em sistemas de e-commerce

### Objetivos
- Detectar problemas comuns em dados operacionais, incluindo:
    - Pedidos sem pagamento
    - Divergências em valores
    - Clientes Duplicados
    - Inconsistências de Estoque
    - Falhas de Integridade Relacional

### Arquitetura
- FastAPI
- SQLite
- Seed com inconsistências
